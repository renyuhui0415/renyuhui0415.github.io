Register a ChatGPT account and Build my chatgpt web

 725 Words  2 Mins  

前言

之前用的账号是学长的账号，前一段时间闲来无事就注册了一下，再加上今天搭建了自己的ChatGPT网站。整理一下，出一个教程，大家有需求的可以按照教程注册账号 / 搭建网站！

注意：需要科学上网，无法科学上网的弄不了哈！

注册ChatGPT账号

选择节点

首先把科学上网的节点切换到美国，避免接收短信后显示无法注册的情况，浪费钱还没有账号。由于我的美国节点挂了，选择了马来西亚节点，可以正常注册！！！

使用Google账号注册

接下来进入 ChatGPT官网，点击Sign up进行注册。

选择邮箱/账号进行注册。这里我选择的用google账号进行注册的

登录成功后会显示如下

绑定手机号

接下来就是绑定手机号了

找一个接码平台来接受验证码，推荐这个 sms接码平台。 自己用邮箱注册一下，然后充值2美金（最低充值），可以使用支付宝。
搜索OpenAI

选择地区为节点地区，我的是马来西亚。

购买后会自动跳转到下面这个页面，复制手机号

返回原来的页面，即可看到验证码，如果不行可以在sms平台换一个手机号或者取消激活。

输入验证码内容，注册就成功了

搭建ChatGPT网站

这个其实就是无脑搭建，在GitHub上找一个开源的ChatGPT项目，然后按照文档部署即可。 我搭建的就是这个开源项目 ChatGPT-Next-Web

点击Deploy部署即可。
注意：这里是使用vercel进行部署，是vercel代理，不需要自己有服务器，但是vercel提供的域名会被墙，需要绑定自己的域名。

接下来需要去获取keys API keys

点击后，输入keys的名字（输不输都行）

记住复制，只会出现一次。
把复制的keys填到刚才的框内，点击Deploy部署即可，等待几分钟就部署好了。

这样就部署成功了
此时可以使用vercel提供的域名访问，如下

输入刚才设置的密码才能正常使用ChatGPT。

现在已经可以正常使用了

由于vercel的域名在国内会被墙，所以咱们绑定一下自己的域名。

点击Settings

点击domains

输入要绑定的域名，会显示错误。

不用担心，去域名控制台配置一下即可 DnsPod

这样表明自己的ChatGPT网站搭建成功了，不用科学上网就能访问。
